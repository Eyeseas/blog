Disallow characters which are made with multiple code points in character class syntax (no-misleading-character-class)

Unicode includes the characters which are made with multiple code points. RegExp character class syntax (/[abc]/) cannot handle characters which are made by multiple code points as a character; those characters will be dissolved to each code point.
